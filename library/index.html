Library

Finbarre Snarey’s Library is a public reference space for documents, guest lists, research notes, media assets and project links connected with my work.

It supports Tarot Interviews, Tarot DMs, the British Tarot Archive and related creative, research and publishing projects.

The library is designed to make useful material easier to find, cite and share. It includes public documents, project summaries, guest lists, links, research outputs, media references and selected assets.

What the library is for

The library acts as the reference layer for my work.

  • Finbarre.com is the main public profile.
  • Project sites such as Tarot Interviews, Tarot DMs and the British Tarot Archive hold the main public-facing material for each project.
  • The library collects supporting material in one place, so press, researchers, collaborators, guests and interested readers can find background information without digging through multiple sites.

What it includes

  • Public documents and reports
  • Guest lists for Tarot Interviews and Tarot DMs
  • Research notes and abstracts
  • Media and citation links
  • Project links and reference pages
  • Images, logos and selected media assets
  • Materials relating to tarot culture, Rider-Waite-Smith practice, British tarot history and living heritage documentation

Main areas

Tarot Interviews

Podcast information, guest references, impact materials and supporting documents connected with the audio interview project.

Tarot DMs

Guest lists, deck references, project information and selected public materials connected with the messenger-based interview series.

British Tarot Archive

Research notes, archive information and supporting material connected with tarot decks, printed material and Rider-Waite-Smith tarot culture in Britain and the UK.

Research and public documents

Abstracts, reports and documentation connected with talks, articles, archive work and living heritage research.

Media assets

Selected images, logos, links and reference materials for press, event organisers and collaborators.
